# Project Power BI - Engeto Academy
This project provides an overview of world countries, cities and languages. Beside a homepage, it is devided into 3 pages, each dedicated to one of these categories. The report highlits the most populated countries and cities worldwide, and it also allows users to apply slicers to explore population rankings within individual continents. Additionally, the visuals present the most common official and unofficial languages and indicate the countries where these languages are used.
